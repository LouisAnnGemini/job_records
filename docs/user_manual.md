# 工作记录台 (Project Archive Management System) 使用说明书

欢迎使用**工作记录台**！这是一个专为项目管理、进度跟踪和资料归档设计的轻量级工具。本说明书将引导您了解系统的各项核心功能。

---

## 1. 项目管理 (Project Management)

系统左侧边栏为项目管理区，支持多项目并行管理。

- **添加项目**：点击左侧边栏顶部的“+”按钮，输入项目名称即可创建。
- **切换项目**：点击项目名称即可切换到该项目的视图。
- **编辑项目**：点击项目名称旁边的编辑图标，可以修改项目名称。
- **删除项目**：点击垃圾桶图标可删除项目。**注意**：删除项目将同时删除该项目下的所有分类和记录，系统会弹出二次确认。
- **项目排序**：通过拖拽左侧项目列表中的项目，可以自由调整显示顺序。
- **侧边栏折叠**：点击顶部左上角的菜单按钮，可以收起或展开侧边栏，为您提供更大的工作空间。在移动端，侧边栏会自动全屏显示。

## 2. 分类管理 (Category Management)

每个项目拥有**独立**的分类体系。

- **进入管理**：在顶部导航栏点击“分类管理”按钮。
- **项目独立性**：A项目的分类与B项目完全隔离。新项目创建时会自动生成默认分类（需求、设计、开发、测试）。
- **自定义分类**：在管理弹窗中，您可以添加新分类、修改现有分类名称或删除分类。
- **分类排序**：在弹窗中拖拽分类条目，可以调整主视图中分类列的左右顺序。

## 3. 记录管理 (Record Management)

记录是系统的核心，支持三种类型：**普通记录**、**待办任务**、**里程碑**。

### 3.1 记录类型
- **普通记录**：用于日常资料归档、笔记等。
- **待办任务 (Todo)**：带有勾选框。可以设置截止日期。
- **里程碑 (Milestone)**：必须设置日期，代表项目的重要节点。

### 3.2 记录操作
- **添加记录**：点击分类列底部的“+ 添加记录”按钮。
- **编辑/查看**：点击记录卡片即可打开编辑弹窗。在弹窗中可以修改标题、内容、类型、日期以及**所属分类**。
- **删除记录**：在编辑弹窗左下角点击“删除”并确认。
- **拖拽排序与移动**：
    - **同分类排序**：在同一列内上下拖拽调整顺序。
    - **跨分类移动**：直接将记录从一列拖拽到另一列。

### 3.3 批量快速添加
- **入口**：点击页面右下角的蓝色“+”悬浮按钮。
- **功能**：
    - 支持跨项目添加记录。
    - **快捷键**：输入标题后直接按 `Enter` 键即可快速保存。
    - **连续添加**：保存后窗口不会关闭，并自动保留上一次选择的项目、分类、类型和日期，方便您连续输入多条记录。

## 4. 特色展示区与视图模式

项目视图顶部有两个固定的展示区域，会自动提取并汇总全项目的关键信息：

- **项目里程碑 (红色区域)**：按时间顺序展示所有类型为“里程碑”的记录。支持全屏查看。
- **待办事项 (蓝色区域)**：按时间顺序展示所有未完成的“待办任务”。支持全屏查看。
- **看板视图**：下方的主体区域，按分类展示所有记录。

## 5. 项目日志 (Project Logs)

系统会自动记录您的关键操作，方便追溯项目历史。

- **入口**：点击顶部导航栏的“日志”按钮。
- **记录内容**：包括增加项目、增加记录/里程碑、完成待办任务、删除记录等操作。
- **独立记录**：每个项目的日志相互独立。
- **筛选与管理**：
    - **筛选**：支持按操作类型（如“增加记录”、“完成任务”）和日期范围筛选日志。
    - **批量操作**：支持全选或多选日志条目，进行**批量删除**或**复制到剪贴板**。

## 6. 待办完成逻辑

系统内置了智能的状态转换逻辑。当您勾选完成一个“待办任务”时：
- **若该任务没有日期**：它会自动转为“普通记录”，并留在原分类中。
- **若该任务设有日期**：它会自动转为“里程碑”，并出现在顶部的红色里程碑区域。

## 7. 数据安全与迁移

- **自动保存**：所有操作都会实时保存到浏览器的本地存储（LocalStorage）中，刷新页面不会丢失数据。
- **导出数据**：点击顶部“导出”按钮，可将所有项目、分类和记录下载为 `.json` 文件进行备份。
- **导入数据**：点击顶部“导入”按钮并选择备份的 `.json` 文件，可恢复或迁移您的所有工作记录。
